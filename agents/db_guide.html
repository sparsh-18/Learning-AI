DB Guide

Tables:

  • users:
    • id (INTEGER PRIMARY KEY)
    • name (TEXT)
    • email (TEXT UNIQUE)
    • password (TEXT)
  • addresses:
    • id (INTEGER PRIMARY KEY)
    • user_id (INTEGER)
    • address (TEXT)
  • products:
    • id (INTEGER PRIMARY KEY)
    • name (TEXT)
    • price (REAL)
  • carts:
    • id (INTEGER PRIMARY KEY)
    • user_id (INTEGER)
    • product_id (INTEGER)
    • quantity (INTEGER)
  • orders:
    • id (INTEGER PRIMARY KEY)
    • user_id (INTEGER)
    • created (TEXT)
  • order_products:
    • id (INTEGER PRIMARY KEY)
    • order_id (INTEGER)
    • product_id (INTEGER)
    • amount (INTEGER)
